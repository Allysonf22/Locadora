Sistema de Locadora de Carros
Sistema de Locadora de Carros desenvolvido com foco em organização, controle e facilidade de uso.
O projeto simula o funcionamento básico de uma locadora, permitindo o gerenciamento de veículos, clientes e aluguéis.

---

 Funcionalidades
 Cadastro de veículos
 Cadastro de clientes
 Cálculo do valor do aluguel
 Consulta de veículos


---
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
